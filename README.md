# 自分用 Twitter ヘッダー

Twitter のヘッダー画像を作り直す。

## サイズ

### サイズの変換方法

実際の大きさ(mm)の 2 倍(px)を四捨五入した値で作成。

面取りの表現で外側の枠線が 2px あるような場合は、その分 2px 小さく作成。
